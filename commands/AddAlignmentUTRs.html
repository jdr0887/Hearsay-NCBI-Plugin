Add Alignment UTRs

Command

ncbi:add-alignment-utrs

About

The ncbi:pull-alignments command only creates EXON instances. Adding the UTR3, UTR5, and INTRON Regions is left to this command. All ReferenceSequences with a NCBI based system identifier are processed. Note that test cases exist for asserting continuity between regions.
